Koneksi Google Spreadsheet

Sambungkan aplikasi ini secara langsung sehingga setiap transaksi pembayaran akan dikirim otomatis ke Google Sheet Anda.

Cara Menghubungkan ke Google Sheet Anda

  1. Buka Google Sheets baru, lalu buka menu Extensions (Ekstensi) > Apps Script.
  2. Hapus semua kode bawaan lalu tempelkan (paste) skrip di bawah ini.
  3. Klik tombol Deploy (Terapkan) > New deployment (Penerapan Baru).
  4. Pilih tipe Web app (Aplikasi Web). Pada bagian "Who has access", pilih Anyone (Siapa saja).
  5. Klik Deploy, lalu salin (copy) **Web App URL** yang diberikan dan tempelkan pada kolom URL di bawah ini.

Kode Google Apps Script:

function doPost(e) {
  try {
    var json = JSON.parse(e.postData.contents);
    var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    
    // Tulis baris header jika lembar spreadsheet masih kosong
    if (sheet.getLastRow() === 0) {
      sheet.appendRow(["Tanggal", "NISN", "Nama Siswa", "Kelas", "Bulan Bayar", "Nominal", "Keterangan"]);
    }
    
    // Tambahkan data transaksi baru
    sheet.appendRow([
      json.tanggal,
      json.nisn,
      json.namaSiswa,
      json.kelas,
      json.bulan,
      json.nominal,
      json.keterangan || "Lunas"
    ]);
    
    return ContentService.createTextOutput(JSON.stringify({"result": "success", "message": "Transaksi berhasil disimpan!"}))
                         .setMimeType(ContentService.MimeType.JSON);
  } catch (error) {
    return ContentService.createTextOutput(JSON.stringify({"result": "error", "message": error.toString()}))
                         .setMimeType(ContentService.MimeType.JSON);
  }
}
